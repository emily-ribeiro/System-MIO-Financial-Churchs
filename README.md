# ⛪ Sistema de Gestão Financeira - ADBRÁS Jaboticabal


## 🚀 Funcionalidades Principais

- **Segurança Multi-Tenant:** Tesoureiros acessam apenas os dados de sua própria congregação, enquanto Administradores possuem visão global.
- **Fluxo de Lançamentos:** Registro de entradas e saídas com categorias padronizadas.
- **Gestão de Parcelamento:** Sistema de lançamentos em cascata (A Prazo) automático para saídas.
- **Auditoria e Fechamento:** Módulo administrativo para revisão diária, tratamento de solicitações de ajuste e fechamento mensal blindado.
- **Painel Gerencial:** Dashboard minimalista com indicadores (KPIs) de saúde financeira e gráficos de distribuição de recursos.
- **Design Minimalista:** Interface focada em UX, responsiva para dispositivos móveis e com navegação intuitiva.

## 🛠️ Tecnologias Utilizadas

- **Framework:** ASP.NET Core Blazor (.NET 9)
- **Banco de Dados:** Entity Framework Core (SQL Server/SQLite)
- **Interface:** Bootstrap 5, Icons Bootstrap e Animate.css
- **Arquitetura:** Scoped CSS (CSS Isolado) para melhor manutenção visual
